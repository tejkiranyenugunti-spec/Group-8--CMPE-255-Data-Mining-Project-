### Project Overview
This project investigates chronic disease disparities across U.S. states by integrating eight public health datasets. We analyze relationships between healthcare spending, workforce capacity, environmental factors (PM2.5 pollution), and behavioral risk factors (physical inactivity) to predict state-level chronic disease burden.

### Objectives
- Integrate and clean multiple public health datasets
- Perform exploratory data analysis on state-level health patterns
- Build predictive models (Linear Regression, Random Forest, Gradient Boosting, XGBoost)
- Apply K-means clustering to identify state health archetypes
- Generate actionable insights for public health policy

### Datasets
1. CDC BRFSS Chronic Disease Dataset
2. Healthcare Expenditure (2015-2020)
3. Physicians Dataset
4. Hospitals Dataset
5. Physical Inactivity Dataset
6. PM2.5 Pollution Dataset
7. Population Dataset (2020 Census)
